Use Cases

Usecase 1: Cross-chain lending marketplace#

Cross-chain lending marketplace is developed based on the ToVchain protocol to enable borrowers to loan cryptos with collaterals on any blockchains without transferring to a single loan exchange. The key features of the cross-chain lending marketplace include:

  • Cross-KYC among diverse blockchain networks to identify borrowers and lenders
  • Mortgaging digital assets on any blockchains without transferring to a loan exchange
  • Loan amount is greater than collateral values
  • AI-based credit score of borrowers using cross-KYC
  • Mortgaging for loans by not only crypto but also all digital assets like NFT and smart contracts
  • Peer-to-peer lending with flexible interest rates defined by borrowers and lenders

Usecase 2: Cross-NFT exchange marketplace#

The decentralized cross-NFT exchange marketplace built up on ToVchain protocol provides an unique way for traders to buy and sell their NFTs created on any blockchain networks. The NFT also can be used as collateral in the cross-chain lending marketplace described in use case 1. In this way, the decentralized cross-NFT exchange marketplace supports NFT Finance (NFTFi) completely. There are several features of this usecase:

  • Cross-KYC among diverse blockchain networks to identify NFT creators, and owners
  • Trading NFT created on any blockchain networks
  • Tracking NFT historical owers on the sea of blockchains
  • Mortgaging NFT assets to loan more

Usecase 3: dApp marketplace#

Using the dApp marketplace built based on ToVchain protocol, developers can create, and sell dApps for users to deploy on different blockchains. The functionalities of the marketplace are:

  • Define once, deploy anywhere quickly and transparently.
  • Cross-KYC among diverse blockchain networks to identify users.
  • Integration across multiple blockchain platforms.
  • Cross migration enables data connectivity across blockchain networks ensuring data security and privacy.

More usecases will be developed in the near future:

  1. Passport 3.0: The passport 3.0 is the decentralized web application built to show profile data of crypto-persons/organizations linked from different sources, which are blockchains and solutions.
  2. Crypto credit score: The credit score application enables to link and evaluate financial and influent data of crypto-persons/organizations in the Internet of blockchain environment.
  3. Fake news detection: The decentralized fake news application allows users to trace news provenance, and thus determine which news is true.
